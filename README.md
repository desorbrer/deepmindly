# deepmindly
An app for tracking and managing psychological issues and conditions

Deepmindly: Mental Health & Self-Regulation Assistant
Deepmindly is a specialized self-monitoring tool designed to help users track their psychological state and prevent burnout through data-driven insights.

The project is built with Laravel and follows a modular approach, allowing for continuous evolution from a classic MVC structure to modern stacks like TALL (Tailwind, Alpine.js, Laravel, Livewire) or Inertia.js + Vue.

🚀 Key Features (Current & Roadmap)
State Tracking: Log daily mood, energy levels, and anxiety markers.

Methodology-Driven Tools: * HALT Method: Quick check-ins to identify if you are Hungry, Angry, Lonely, or Tired.

Broken Windows Theory: A system to track small "messy" areas in life/habit that might lead to a larger mental decline.

Analytical Dashboards: Visualizing emotional trends over time.

Refactoring-Ready Architecture: Designed with clean code principles to facilitate future transitions to SPAs or reactive interfaces.

🛠 Tech Stack
Backend: Laravel 13 (PHP 8.3+)

Database: MySQL

Frontend: Blade / Tailwind CSS (Roadmap: Migrating to Inertia + Vue / Livewire)

Tools: Laravel Pint (Style), Pest (Testing)

📈 Evolution & Learning Path
This project serves as a "living" codebase. My goal is to demonstrate:

Code Evolution: Moving from monolith to reactive components.

Best Practices: Implementing Service Layers, Form Requests, and DTOs as the project scales.

Domain Driven Design (Elements): Applying clear logic to psychological assessment modules.
